# DirectDrop

DirectDrop is a web-based peer-to-peer file sharing application designed for fast and private transfer between devices. It allows people to connect through a shared room link or QR code and exchange files directly using browser-to-browser communication.

The project focuses on simple sharing flows for both one-to-one and group scenarios. It includes real-time connection signaling, room management, and a modern interface that works across desktop and mobile browsers.

This app is built to make file sharing quick without requiring account creation, while keeping the transfer experience smooth and practical for everyday use.

## What this project provides

- Direct device-to-device file sharing
- One-to-one and broadcast room modes
- QR-based room joining for faster access
- Optional room protection with password support
- Real-time connection status and transfer feedback
- Progressive Web App behavior for installable usage
